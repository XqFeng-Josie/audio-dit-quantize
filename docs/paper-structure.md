## 0. 一句话主线

> TTS DiT 的 W4A4 代价高度各向异性——**文本稳健、代价特异地落在音色**,且音色代价有任务特有的**模块×时间步结构**(FFN×晚步,时间方向与视觉先验相反)。把"任务感知"注入 PTQ 有三个可选层——**数据层(选什么标定)、目标层(标定优化什么 loss)、分配层(精度给谁)**——本文逐层实测:**样本级选数据统计上先天盲且不跨尺度**(数据层内唯一可迁移杠杆是快照的时间步放置);**重建目标的感知重加权 0/6 全败——连任务度量的真梯度引导(percw)也干净 ns——且有梯度显著性机制解释**(任务显著坐标 ⊥ 重建难点/离群坐标;瓶颈在注入层、不在引导信号);**只有等预算的结构对齐分配双尺度显著成立**。→ **任务感知必须注入在结构/分配层,不在样本打分或损失重加权层。**

**定位**:系统研究(diagnosis + 逐层判定 + 存在性证明),不是 SOTA 刷表方法论文。分配配方(late_ffn/iso_late)是"结构可分配"的 proof-of-concept,增益温和、随尺度递减,**不作性能卖点**。

**叙事流**:设置(含方法基线表:RTN 崩溃 → W4A4 是真悬崖)→ **诊断**(代价在音色、双源、FFN×晚步、半反视觉)→ 三层注入逐层判定:**数据层**(先验池+任务指标筛有效,但无便宜代理;快照时间放置是唯一因果轴)→ **目标层**(重加权全败 + 机制)→ **分配层**(等预算成立,climax)→ 讨论。

### C1 · 标定数据:从先验池设计到系统判定(方向 1,建构式四步叙事)

- **① 池设计(先验驱动)**:按测试集形态镜像设置候选池 pool_v1——prompt **时长**与**文本长度**按 Seed-TTS 分位设接收窗(修剪后测量);**响度**逐条记录;**说话人** 158 人、性别轮转均衡、说话人级隔离;**文本风格**镜像语言构成(zh_normal / zh_hardlike 三亚型 / en);**中文音节结构**由确定性生成器按 hard 分类学覆盖(混淆声母韵母 / 重复 / 长拼接);全部集外 + 交叉配对 + 五项污染审计。间接效果证据:从该池随机抽 32 条,W4A4 文本成本已压到全 ns(1B 全测集 2×2)——**干净镜像池 + 随机抽样本身就是强默认**。〔无"坏池"对照臂:只能写协议 + 间接证据,不写因果"池先验有效";要因果需补消融,§6-P2〕
- **② 集合级筛选(任务指标,非 proxy)**:10 随机集好坏差异真实(hard 散布 = 平均成本 5 倍);按六轴综合排名筛出最优批 {s4,s7,s9} vs 最差 {s5}——固定尺度有效;跨尺度**部分保持**(s7 #3→#1、s9 #2→#2、s5 #10→#10;综合 rho=0.49、n=10 未显著;s4 #1→#4 是反例)——写"部分保持",不写"迁移成立"。

### C2 · 任务敏感度:归因 → 结构 → 等预算验证(方向 2;③④⑤ 三级因果链)

> 取舍定调(2026-07-23):C2 本体只扛 **③归因 / ④结构 / ⑤等预算验证** 三块;**①各向异性压缩进 §3 末段(Finding 1 + 主表)、②声音尾部降为一句话+附录**——①不能全砍:③④⑤ 全部以 ΔSIM 为主指标、CER 仅作守卫,该设定的合法性来自"代价特异落在音色";②是"效应量小"质疑的标准回击,留一句牙齿。两者合计 ~0.3 页。

- **前置(进 §3,~0.3 页,不占 C2 本体)**:各向异性(SIM 三轴一致向下、多数 SIG;CER/WER 符号不定、多数 ns;自然度不伤;音色从 8-bit 就伤、无 8/4 二分)+ 尾部一句(固定 10% 说话人掉 5–10×、跨尺度保持、性别公平;明细附录)。
- **归因(③,T2)**:**SIM 双源**——权重+激活各自 SIG、近似相加,跨尺度成立、3.5B 幅度减半 → **激活份额是可分配的抓手**;CER 唯一 SIG 例外(3.5B-zh)是**权重×激活交互格**(W16A4/W4A16 单独 ns、组合 SIG),文本整体稳健。
- **结构(④,F3)**:音色的激活侧成本住 **FFN**(保 FFN>保自注意力,双尺度 SIG,与视觉 TaQ-DiT **一致**)× **晚步**(双尺度 SIG,与视觉时间步先验**相反**;标定快照轴 L2 独立复现——同一反转两处证据);"量化自注意力在 hard 有益"划界一句。
- **等预算验证(⑤,climax,F5)**:**iso-budget(预注册)**——平均位宽严格钉 4.0 只做重分配:**时间轴 iso_late−iso_early 双尺度三集 6/6 SIG**(效应为加预算版 3–5 倍);模块轴 1B 三集 SIG、3.5B 带 0.105-bit 预算劣势 en 仍 SIG;**3.5B iso_late 带失配劣势仍显著超 uniform A4(存在性证明)**;**押错方向(iso_early)比 uniform 更糟**——方向双向都要紧;文本 36 项仅 1 边缘。外加预算-音色权衡曲线(late_ffn 17% / ffnfp 50% / noact 100%),late_ffn 为高性价比低预算点、非天花板。
- **证据状态**:最硬的一根柱子,双尺度、全测集、预注册。
- **划界**:增益温和且随尺度递减——作"结构真实可分配"的证据,不作性能卖点。
- 溯源:总编 §2.1–2.3(前置)、§3.1–3.3、§4。

### C3 · 标定目标:重建 ⊥ 感知——瓶颈在注入层,不在引导信号(方向 3;定稿口径,2026-07-24)

- **主张(定稿)**:把标定重建目标向任务损伤位置重加权的**整族干预系统性失败**——计分板 **0/6**,覆盖**五条重加权轴**:空间(M2 生成区 / L-A prompt 区)、通道方差(M3)、通道感知梯度(L-C percw)、时间快照(L-B)、深度权重(W1);其中**三次"错方向主动有害"**(M3 伤 SIM、L-A 伤 en SIM、L-B 伤 hard CER),W1 给出**重建保真度 ⊥ CER** 的干净解耦(最难块重建 loss 腰斩、hard CER 纹丝不动)。
- **升级论点(L-C 的价值)**:**连任务度量的真梯度引导也买不到感知**——L-C 用 WavLM 说话人 cos-sim 反传的逐块×逐通道显著性作权重,结果干净 ns(6 项主指标全 ns、守卫无命中、尾部不超零对照,六臂中唯一无主动伤害)→ **失败不是"信号选错"能解释的,是"逐通道/逐区域重加权重建目标"这个注入层表达不出感知增益**。启发式信号会主动伤对侧轴、真梯度信号无害但无增益——两档结局共同钉死注入层。
- **机制(把负结果升级为发现,双尺度复现,总编表 6-S1/S2/S3)**:
  - ① **梯度重要性与重建难度深度剖面反相关**(Spearman −0.94/−0.79):标定目标最卖力的深块恰是梯度重要性最小的块 → W1 失败的机制层解释;〔诚实划界:energy 对照同样单调,此剖面是**通用解码梯度结构**,不得表述为"说话人感知住浅块"〕
  - ② **显著通道 ⊥ 方差通道 ⊥ 离群通道**(top-100 重叠在随机水平:spk∩方差 4/6、spk∩amax 3/3,两尺度):量化机制天然围着离群/方差通道转,感知重要性在别处 → M3 伤 SIM 的机制 + "W4A4 为何伤音色"的候选解释;
  - ③ 特异性对照(energy):spk/energy 剖面 Spearman +0.91/+0.94、top-100 重叠 69/67——通道剖面**非说话人特异**,降级为"通用解码承载通道"(跨深度稳定 +0.97、中等集中)——照实写,也解释 percw v1 为何无增益(特异分量未分离)。
- **方法学附赠(写一句,细节附录)**:尾部恢复评估的**零对照设计**——s9 随机 draw 作回归均值控制(零对照自身 zh/en"显著恢复"+0.019/+0.030,三臂均不超它)——没有它,尾部指标会把回归均值误读为效应。
- **收束**:感知/任务信号买不进重建目标的重加权,**必须注入分配层(C2⑤)与数据层的时间放置(C1⑤)**——三层判定闭环,直接支撑主线。
- **残臂(可选,不改口径)**:L-C v2(对比显著性 sal_spk−sal_energy 分离说话人特异分量;离线零新计算、一次标定+gen)——若成立作 bonus 段/未来工作,不重开决策门。
- **证据状态**:负向臂 1B(dev)+ 机制双尺度;L-C 3.5B 在跑(判负臂按预注册无须跨尺度,回来若同样 ns 可把划界升级为"最强臂双尺度复现")。
- 溯源:总编 §5、§6(表 6-R1–R4、6-S1–S3)。

### 贡献 bullets(引言用,4 条)
