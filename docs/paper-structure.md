> **标题**:Task-Structure-Aligned Precision Allocation for Quantizing Text-to-Speech Diffusion Transformers
> 目标:AAAI-27(7 页);证据全部指向 `results-consolidated.md`。原则:**只放强且干净的结论,砍掉价值低/易被问穿的**。

## 0. 一句话主线

> TTS DiT 的 W4A4 代价高度各向异性——**文本(可懂度)整体稳健,代价主要落在音色(SIM)**。音色成本来自**权重和激活两者**(激活的损伤**在 1B** 只去 SIM、不去 CER;**3.5B 归因尺度依赖**——激活反而驱动小 zh-CER 代价);其中**激活侧的音色成本有子模块×时间步结构**——住 FFN、且晚步更关键——可用**结构对齐的精度分配**回收。这个结构**一半与视觉一致**(FFN 敏感,TaQ-DiT 佐证)、**一半与视觉相反**(护晚步而非早步);而**数据侧选标定虽能在固定尺度挑出较优集、却不跨模型迁移**(即使同族)。→ 精度必须按任务实测的结构来分配。

**定位(A 框架)**:论文以**诊断**为主体——代价在音色、视觉先验半错、数据选择不迁移;**结构对齐的精度分配是这个原则的 proof-of-concept**(保 FFN,增益温和、随尺度递减),不作性能卖点。

**叙事流**:设置+池 → **数据选择(固定尺度能挑最优、但不跨模型迁移)** → **退化解剖(CER/SIM 都伤,伤在哪?)** → **归因(CER 无一致结论、SIM 双源;定位到 FFN×晚步)+ 结构对齐分配(climax)**。数据侧的"选不动"是结构侧贡献的动机,分配放最后做高潮。

## 1. 三根主柱(正文只扛这三根;每根双重证据 + 主动划界)

> 收敛原则:原 A 级 7 条 → **3 根柱子**,每根都有**双重独立证据**、尽量**双尺度**;其余全部为柱子服务或下沉。**不堆结论、不碰"效应量大小"、单家族主动划界。**

### 柱一 · 代价特异地落在音色(不是文本、不是自然度),且可感知
- **各向异性(判据=方向一致性+显著性)**:SIM 三轴 vs fp32 **一致向下、多数 SIG**(双尺度)= 可靠代价轴;CER/WER **符号不定、多数 ns**(文本稳健;唯一 SIG 例外 3.5B-zh);**自然度 UTMOS/DNSMOS ≈ fp32**〔MOS 待并〕。→ 三维里只有音色是可靠代价。
- **感知牙齿(回击"0.003 可忽略")**:平均 SIM 掉 0.003 是均值假象——**固定 10% 说话人掉 0.02–0.04(5–10×)、跨尺度保持、性别公平**(§1.4 声音尾部,**提正文**)。
- **位宽**:从 8-bit 就伤,无干净 8/4 二分。
- 证据:§1.1 / §1.2 / §1.4 /〔MOS〕。

### 柱二 · 音色代价有"模块×时间步"结构,一半与视觉相反(双路确认)
- **双源 + 抓手**:SIM 被权重和激活两者伤(近似相加);**激活那份是可分配的抓手**(§2.1)。
- **结构 = FFN × 晚步**:保 FFN>保自注意力(**双尺度 SIG,与视觉一致**,TaQ-DiT 佐证);晚步>早步(**双尺度 SIG,与视觉相反**)。
- **"晚>早"两条独立轴确认**:① 精度分配(§2.2)② 标定快照(§4.4/L2,**提正文**)——一个反直觉结论、两处独立证据,极难反驳。
- **构造性证明**:保 FFN 激活回收激活侧音色成本(双尺度 SIG,增益温和;**proof-of-concept,不作性能卖点**)。
- 证据:§2.1 / §2.2 / §3.1 / §4.4。

### 柱三 · 不能照搬,必须实测(三个独立的"不迁移")
- **视觉先验半错**:晚步翻转(柱二)。
- **数据选择不跨尺度**:集合好坏 **rho=0.03**(10 draw × 2 尺度);语言规则不迁移(1B 显著伤 zh → 3.5B 消失)。
- **标定目标 ⊥ 任务**:W1 腰斩最难块重建 loss、hard CER 不动 = **重建⊥CER**(1B);**深度/重建轴分配失败、任务实测的模块×时间步轴才有效**。
- → **精度必须按任务实测的结构分配(= 标题)**。
- 证据:§4.2 / §4.3 / §3.2。

### 支撑与取舍
- **提正文加固柱子**:声音尾部(§1.4→柱一)、L2 晚>早(§4.4→柱二)。
- **保留但严格划界、不承重**:W1"重建⊥CER"(明标 **1B-dev**)、语言(一小段例证)、位宽/双源。
- **降附录**:错误分类学(重复结构)、深度剖面明细、M2/M3、cross-attn(null)、10 子集全表、跨尺度排名。
- **不进论文**:top6<bot6"深块噪声有益"(单 draw)、"量化噪声有益"暗线、把 FFN 包装成"与视觉相反"(TaQ-DiT 会反驳)。

### 主动划界(preempt 审稿)
- **单家族**(LongCat 1B/3.5B):卖点是**跨 3.5× 尺度复现** + **TTS-vs-视觉跨域**对比;跨架构留未来。**论文不依赖 F5**。
- **效应量小**:只用方向一致性+显著性判据,不碰"大小";感知性由**声音尾部 +〔MOS〕**承担。
- **fake-quant**:引 FlatQuant kernel 得真 4×;质量分析与 kernel 效率正交。

## 2. 章节规划(7 页)

### §1 引言(1.0 页)〔暂不完整,先搭骨架〕
- 动机:TTS 部署;**8-bit 已近免费、4-bit 才是真正的边界**。
- 空白:图像/视频 DiT PTQ 有系统研究,**TTS 从未有 W4A4 系统研究**。
- 贡献四条:① 首个 TTS DiT W4A4 系统解剖——**代价在音色不在文本**,音色**双源(权重+激活)**;② **音色的激活侧成本有 FFN×晚步结构**(可分配的抓手)——据此**保 FFN 激活回收音色的激活侧成本**(结构对齐分配的 proof-of-concept,增益温和、随尺度递减);③ **一半借力视觉、一半与视觉相反**(FFN 敏感一致 / 护晚步相反);④ **数据侧选择不跨模型迁移**(固定尺度能挑最优、但集合好坏跨尺度 rho=0.03;唯一显著的语言效应到大模型消失)。

### §4 标定数据:能靠选数据解决吗?(0.75 页)——数据侧,失败即动机
- 建好干净池后自然的第一问:**选标定数据能改善 W4A4 吗?** K=10 随机集 → 指标随数据大幅波动、固定尺度上确有较优/较差之分。
- **两条硬结论 = "选不动 / 不跨尺度"**:① 集合好坏跨尺度 **rho=0.03**(1B 最优 s8 在 3.5B 掉到中游、1B 倒二 s0 在 3.5B 第一);② **各构成/模型侧因素**(难例配比、规则集、初始损失打分等)**全负**,无一能预测或挑出好集。
- **一小段例证(语言构成,不展开、不头条)**:唯一有 1B 因果效应的因子——语言构成——1B 混英文**显著伤 zh**(+0.319 SIG);但它**不跨尺度**:3.5B 上这个显著代价**消失**(en32−en0 −0.147 ns、非单调)。连唯一的真因子都是尺度特异的。
- 收束:**数据选择即使同族也不可靠;退化有结构,必须直接处理结构** → 引出 §5/§6。
- 〔K=10 指标数值在此引入,§5 深挖其结构;不重复堆表。〕

### §5 退化的结构(1.0 页)——解剖:代价一致地落在音色
- **T1 主表**:fp32 / W8A8 / W4A4 × zh/hard/en × CER+SIM × 1B/3.5B。
- **各向异性(判据 = 方向一致性 + 显著性,直接读 T1)**:**SIM 三轴一致向下且多数 SIG → 音色是可靠的代价轴;CER/WER 符号不定、多数 ns → 文本更稳健**(有时反而更好)。**hard CER 幅度虽最大但方向不定、多数 ns;唯一 SIG 例外 3.5B-zh CER +0.11**,不构成三集一致代价。**判据是方向一致性、非变化大小**(SIM 绝对值虽小,但方向稳)。(A 级 1)
- **位宽(A 级 2)**:音色从 8-bit 就伤(W8A8 SIM 已 SIG);文本无干净的 8/4-bit 二分。
- **音色损伤 = 普遍 + 脆弱声音尾部**(B 级一段:5–10× + 性别公平)。
- 尺度:一句(成本稳定/散布收缩/**zh 红利消失、3.5B-zh CER 反受损**/SIM 减半)。→ 代价压在音色,下节归因"音色住哪、怎么分配"。
- 〔文本侧 §5 只说"**稳健**";"重复结构失稳"是 dev 错误分类学的细化(T1 不展示严格 CER),**降附录、§5 不主张**。〕

### §6 归因与结构对齐分配(2.0 页)——核心 / climax
- **归因 2×2(T2)**:fp32 / W16A4 / noact / full × CER+SIM(全测集同一评测集)。① **CER 归因尺度依赖**:1B 无一致显著归因(激活⊥CER、权重侧小);**3.5B 出现小 zh-CER 代价、由激活驱动**(待 W16A4 坐实);② **SIM 被权重和激活两者显著伤**(noact/W16A4 各自 SIG、近似相加)。→ 音色成本双源,**激活那一份是可分配的抓手**。
- **任务特征 ↔ module(T3 / F1)**:音色收益**住 FFN**(保 FFN>保自注意力,**双尺度** SIG,**与视觉 TaQ-DiT 一致**)× **晚步更关键(双尺度,与视觉相反)**——且**标定快照轴(L2,§4.4)独立复现"晚>早"**,同一反转两处证据。cross-attn 中性作次要补充。
- **结构对齐分配**:保 FFN 激活回收音色的激活侧成本,**预算-音色权衡曲线**(late_ffn 17% / ffnfp 50% / noact 100%),late_ffn 高性价比低预算点(A 级 5)。**增益温和、随尺度递减——作"结构可分配"的证据,不作性能卖点(A 框架)。**
- **两条分配轴的对照(直接论证"task-structure-aligned")**:**深度轴**(重建 loss 驱动、视觉/LLM PTQ 常规先验,§1.5)——照它分配位宽 = W1 = **失败**(最难块重建腰斩、hard CER 纹丝不动 → **重建⊥CER**;1B;**不写 top6<bot6**);**模块×时间步轴**(任务实测:FFN、晚步)——**才是有效轴**。**naive 重建先验在 TTS 上是陷阱**;可懂度在 per-block W4 下不可约(1B)。(A 级 6)

### §7 讨论 / 限制 / 结论(0.5 页)
- **一借力一翻转**:结构对齐分配一半借视觉的力(FFN 敏感)、一半与视觉相反(护晚步);加**数据选择不跨模型**——两个"不可整体搬运",都指向"按任务实测的结构分配精度"。
- 限制(主动披露):单模型家族(F5-TTS 待补)、fake-quant、MOS 代理无真人听测;CER 归因不完整(尺度依赖)。
- 实践指南:W4A4 TTS 别在标定数据上花力气(不跨模型);**保 FFN 激活是可靠的音色杠杆**;文本对 W4A4 整体稳健。
